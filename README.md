# Very Small MIPS Interpreter
A MIPS interpreter(simulator) written in Rust.

## Supported
- [ ] Arithmetic, Logic
    - [x] ADD
    - [x] ADDI
    - [x] SUB
    - [x] MUL
    - [x] DIV
    - [x] AND
    - [x] ANDI
    - [x] OR
    - [x] ORI
    - [x] XOR
    - [x] XORI
    - [ ] NOT
- [x] Constant
    - [x] LI
    - [x] LUI
- [ ] Comparison
    - [ ] SLT
    - [ ] SLTI
    - [ ] SEQ
    - [ ] SGE
    - [ ] SGT
    - [ ] SLE
    - [ ] SNE
- [ ] Branch
    - [ ] B
    - [x] BEQ
    - [ ] BGE
    - [ ] BGT
    - [x] BLE
    - [x] BLT
    - [ ] BNE
    - [ ] BNEZ
    - [ ] SLT
- [x] Jump
    - [x] J
    - [x] JAL
    - [x] JR
    - [x] JALR
- [ ] Load, Store
    - [ ] LA
    - [ ] LI
    - [x] LW
    - [x] SW
- [ ] Transfer
    - [x] MOVE
    - [ ] MFHI
    - [ ] MFLO
    - [ ] MTHI
    - [ ] MTLO
    - [ ] MOVN
    - [ ] MOVZ
- [ ] Exception, Interrupt
    - [ ] SYSCALL
        - [x]  1: print_int
        - [ ]  4: print_string
        - [x]  5: read_int
        - [x] 10: exit
    - [ ] SYSCALL(own syscall)
        - [x] 11: print_int + '\n'
    - [ ] BREAK
    - [x] BOP
